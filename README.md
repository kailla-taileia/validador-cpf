**Validador de CPF em JavaScript**
------------------------------------

Este projeto é um validador de CPF desenvolvido em JavaScript, utilizando lógica de programação, manipulação de arrays e funções para realizar o cálculo dos dígitos verificadores.
O usuário insere um CPF, e a aplicação retorna se ele é válido ou inválido de acordo com as regras oficiais de validação.

**📌 Funcionalidades**
Verificação completa dos dígitos verificadores
Cálculo automático dos pesos de multiplicação
Interface simples para inserir e validar o CPF
Retorno claro informando se o CPF é válido ou não

**🛠️ Tecnologias Utilizadas**
HTML5
CSS3
JavaScript (ES6+)

**📚 O que aprendi / pratiquei**
Uso de métodos como map(), reduce() e splice()
Criação de funções puras e reutilizáveis
Manipulação de DOM
Estruturação de um projeto simples em JavaScript
Lógica para calcular os dígitos verificadores de um CPF
